# Issue: Create a Python hash report script

## Summary

สร้างสคริปต์ Python สำหรับคำนวณ hash และแสดง metadata เบื้องต้นของไฟล์

## Why this matters

- เป็นเครื่องมือพื้นฐานที่ใช้บ่อยในการวิเคราะห์ไฟล์
- เหมาะกับระดับเริ่มต้น
- ใช้ต่อยอดเป็น report ได้ทันที

## Tasks

- รับ path ของไฟล์จาก command line
- แสดงขนาดไฟล์
- แสดง MD5, SHA1, SHA256
- เขียนโค้ดให้อ่านง่ายและเหมาะกับผู้เริ่มต้น

## Definition of done

- มีไฟล์ `analysis/scripts/hash_report.py`
- รันกับไฟล์ตัวอย่างได้
