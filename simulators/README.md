# Simulators

โฟลเดอร์นี้ใช้สำหรับตัวจำลองที่ไม่เป็นอันตรายเพื่อการศึกษาเท่านั้น

ตัวอย่างที่อนุญาต:

- การสร้าง event ปลอมเพื่อทดสอบ detection
- การจำลอง process แบบ logging-only
- การสร้าง file activity แบบปลอดภัยในโฟลเดอร์แล็บที่กำหนดไว้

ตัวอย่างที่ไม่อนุญาต:

- การเข้ารหัสไฟล์ผู้ใช้จริง
- กลไก persistence
- การดักหรือขโมยข้อมูลรับรอง
- พฤติกรรมหลบเลี่ยงการตรวจจับ
- network beacon ไปยังปลายทางที่ไม่ได้ควบคุม
